Task: VERBEUBT.A2
The main purpose of this activity is to map the user request of the migrated application to a user request in the non-cloud application. To mirror the interaction of the user with the migrated application, all user requests must also be executed on the non-cloud application. Therefore each discovered request must be mapped to an equal request on the non-cloud application. Due to the abstractions done in the reverse engineering step and the specializations and refinements performed in the forward engineering step, a simple mapping might not be possible. Every change on the PSM and PIM level must be considered to allow for an efficient mapping of which non-cloud component corresponds to which migrated component and more importantly how the input for the methods must look like.
