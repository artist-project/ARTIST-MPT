Task: OPTAPPFEA.A1
The modeller selects a concrete non-cloud PIM view that describes a non-cloud feature implementation he/she wants to modernise. The Optimization Tool (OT) inspects the model view and its associated requirements and identifies the suitable cloudification patterns (available in the Repository) expressed as transformations. In case these transformations can be parameterised, the user can be requested by the OT to fulfil these parameters. Alternatively, the OT can fulfil them using the requirements associated to the model view. Then, these cloudification patterns are applied in a transformation chain. As a result a cloudified PIM view is obtained.
