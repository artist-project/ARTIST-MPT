Task: TECHEVA.A1
The security and risks management analysis activity focuses on the analysis of the management of security and risks associated with the development and delivery, of cloud based applications. Cloud computing systems must address security requirements such as authentication, authorization, availability, confidentiality, identity management, integrity, audit, security monitoring, incident response, and security policy management [2] [3]. The security and risk management analysis requires an analysis on the security capabilities for cloud services, including access control, confidentiality, integrity and availability. For this, a set of security specific components must be included as part of the application or as external third party component. These include:
o Authentication and identity management component
o Authorization & Service Policy Management component
o Data protection component.
