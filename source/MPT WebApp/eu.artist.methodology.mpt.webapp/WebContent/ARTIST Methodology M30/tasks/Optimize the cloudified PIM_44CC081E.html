Task: OPTAPPFEA.A2
The modeller selects a concrete cloudify PIM view that describes a cloudify feature implementation he/she wants to modernise. Moreover, he/she also selects the target Cloud environment he/she wants to optimize this implementation for. The Optimization Tool (OT) inspects the model view and its associated requirements and identifies the suitable optimization patterns (available in the Repository) express as transformations. In case these transformations can be parametrised, the user can be requested by the OT to fulfil these parameters. Alternatively, the OT can fulfil them using the requirements associated to the model view. Then, these optimization patterns are applied in a transformation chain. As a result, an optimized PSM view is obtained.
