Task: FINANCIAL.A3
The objective is to define the pricing strategy taking into consideration each customer segment. There are several possible license models.
- By subscription
- By use
- By transaction
- By value
- Flat rate
- Hybrid
The most appropriate one for being cloud compliant are: by use or by transaction. Based on the license model different parameters should be measured. These parameters could be:
- Time: the time that the client uses the service measured in hours, seconds or microseconds.
- Volume: The volume of the used service: GB, MB/s, CPU cycles, etc.
- Sessions: number of accesses to the service
- Requests: number of requests to the system
- Users: number of users that are allowed to access to the system
