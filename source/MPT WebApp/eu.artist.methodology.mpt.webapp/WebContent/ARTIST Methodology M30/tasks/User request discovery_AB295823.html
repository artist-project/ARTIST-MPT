Task: VERBEUBT.A1
The main purpose of this activity is to discover real user interactions with the modernized system. This can be understood as the retrieval of the interaction between the user and the migrated software in the form of user requests. One request of the user usually corresponds to one action on the cloud side, e.g., the retrieval of data from the database or the manipulation of objects used by the application. User requests might be discovered either directly when the user sends a request to the migrated application or are retrieved from already established log-files.
