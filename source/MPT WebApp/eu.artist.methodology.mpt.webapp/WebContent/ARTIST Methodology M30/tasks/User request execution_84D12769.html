Task: VERBEUBT.A3
The main purpose of this activity is to execute the user requests both in the non-cloud application and in the modernized application. While the execution of the migrated request is performed by a Real User, the non-cloud request is performed by a Virtual User. What this means is that requests to the migrated app are made by an authentic user and requests performed on the non-cloud app are made by an artificial user
