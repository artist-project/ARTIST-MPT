The purpose of this task is to obtain the feedback of the user regading the offered services.

        Main Description
        In order to improve the user satisfaction it is important to collect their feedback. There are three main actions:
        - Collect the feedback of the users according to the process defined on the communication mechanism (satisfaction measures).
        - Analyze the received feedback.
        - Provide solutions or improvement to cover the problems detected by the satisfaction measures.
