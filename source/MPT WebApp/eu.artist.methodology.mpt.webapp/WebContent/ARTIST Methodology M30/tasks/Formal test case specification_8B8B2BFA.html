Task: VERBETC. A1
The test selection criteria represent the test purpose of the testing process. These criteria will guide the generation of the test cases in the testing tool. The PIM and the migration goals are analysed in order to derive specific test selection criteria, such as certain structural model coverage criteria and data coverage criteria.
