Task: MCRP.A1
This activity should define the mechanism for communicating with the user, the way in which this mechanism is going to interact with the user (email, Phone, sms...) and the way to measure the satisfaction of the users, collecting their feedback and analyzing it.
Disciplines: Process Tasks

    Outputs
    • Communication Mechanism
      Main Description
      This mechanism sould cover the following characteristics:

      - accessible

      - It offers a uniform service of consistent quality.

      - It keeps users regularly informed and logs all interaction with them.

      - It provides support for the business.

      - It measures the satisfaction of the users

      - It analyses the feedback of the user
      Steps
      Define the mechanism for communicating with the user
      Define the way in which this mechanism is going to interact with the user (email, Phone, sms...)
      Define the way to measure the satisfaction of the users, collecting their feedback and analysing it.
