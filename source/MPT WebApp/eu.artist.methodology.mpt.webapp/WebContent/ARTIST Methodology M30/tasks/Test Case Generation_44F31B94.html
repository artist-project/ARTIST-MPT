Task: VERBETC. A3
After the test model and test cases specification have been created, a combination of test case generation approaches is applied in order to generate abstract test cases, including test input data and oracle functions. These abstract test cases can now be translated into executable test scripts. Therefore, first test scripts for testing the migrated PSM on model level are derived before deriving test scripts that also allow for testing the forward-engineered migrated software.
