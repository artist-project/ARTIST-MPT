Task: SCTARGET.A2
The modeller requests RST to search for candidate Cloud environments that fulfil the requirements expressed on the non-cloud PIM view he/she has selected. RST initiates a matchmaking and ranking process that inspects the available Cloud environment models (in the Repository) looking for matches between requirements and Cloud environment services/features. Results are presented to the user ranked by the precision in the matching
