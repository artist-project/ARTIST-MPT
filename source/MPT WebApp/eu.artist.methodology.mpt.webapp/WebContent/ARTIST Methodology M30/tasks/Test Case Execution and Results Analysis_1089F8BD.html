Task: VERBETC. A4
This activity should execute the test cases and analyse the test report. This test report could contain an error description on model level (and not only on source code level), which seems to be more helpful for fixing the respective issue in the migration of the PIM to the PSM. If the tests succeed on model level and only break in the migrated software, an error trace only on source code level is obtained; but in this case, the issue probably lies in the forward engineering components that generate the source code (e.g., source code generator templates), so the information on the source code level is appropriate for fixing such an issue.
