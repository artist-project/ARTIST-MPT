Task: VERBEUBT.A4
The main purpose of this activity is to discovery the necessary runtime traces from both applications. Having executed correspondingly equal requests on both applications, non-cloud and migrated, we can check whether they behave correspondingly equal by comparing their runtime traces. However, before that these platform specific traces need to be extracted both from the non-cloud application and the migrated application. That is exactly what occurs in the “Runtime trace extraction” activity. A platform specific runtime trace might include method calls with their respective parameter values and/or depending on the type of request and application an output value.
